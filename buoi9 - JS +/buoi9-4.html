2. Toán Tử Logic

    'And(&&)' 
        vd: expr1 && expr2 : trả về true nếu expr1 và expr2 đều là true
    'Or(||)'
        vd: expr1 || expr2 : trả về true nếu expr1 hoặc expr2 đều là true
    'Not(!)'
        vd: !expr          : trả về giá trị false nếu biểu thức đúng và trả về giá trị true nếu biểu thức sai
