1. Câu Lệnh Điều Kiện IF - ELSE - ELSEIF

    if(biểu thức điều kiện) {
        < khối lệnh if >
    }

2. Câu Lệnh Điều Kiện If- Else

    if(biểu thức điều kiện) {
        < khối lệnh if >
    } 
    else {
        < khối lệnh else >
    }

3. Câu Lệnh Switch - Case

    switch(biểu thức) {
        case < giá trị so sánh 1 >:
         < mệnh đề 1 >;
         break;
        case < giá trị so sánh 2 >:
         < mệnh đề 2 >;
         break;
        case < giá trị so sánh 3 >:
         < mệnh đề 3 >;
        ...
        default: < mệnh đề mặc định >
    }

4.Câu lệnh vòng lặp

    - Thực hiện khối lệnh lặp đi lặp lại đến khi gặp điều kiện dừng
    - các câu lệnh vòng lặp
        + for // khởi tạo - điều kiện - khối lệnh - tăng/giảm trị
          for(< khởi tạo >;< điều kiện >;< tăng/giảm trị >){
            < khối lệnh >
          }

        + do while
          do {
            < khối lệnh>
          } while < điều kiện >
          
        + while
          while(< điều kiện >){
            < khối lệnh >
      }
